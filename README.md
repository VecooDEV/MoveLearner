* Support Pixelmon and Cobblemon mods.
* Convenient system for studying moves on pokemon.
* Support for filters and search for moves.
* Possibility to set your own prices for moves.
* GUI with support for various resource packs.
* Almost completely customizable mod.

![](https://i.imgur.com/Lc7kBGj.png)
* Select a pokemon, and in the move menu look for the necessary move on the pokemon, the entire list of moves is taken from the information about the pokemon, so if desired, the list can be expanded, and then confirm the purchase of the move.
